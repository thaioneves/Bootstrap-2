Justify Content - Alinhamento Horizontal

Usada dentro de row ou d-flex.

Valores:

justify-content-start: alinhado à esquerda

justify-content-center: centralizado

justify-content-end: alinhado à direita

justify-content-between: espalhado (inicio e fim)

justify-content-around: espalhado com espaço igual

Align Items - Alinhamento Vertical

align-items-start: alinhado no topo

align-items-center: centralizado verticalmente

align-items-end: alinhado na base

Posicionamento

O Flexbox é uma tecnologia do CSS usada para organizar elementos em linhas ou colunas de forma simples e responsiva.

Position Static (Padrão): O elemento aparece exatamente onde foi colocado no HTML.
Position Relative: O elemento continua ocupando seu espaço original, mas é movido 50px para a direita.
Position Absolute: O elemento fica preso ao canto superior esquerdo do elemento pai.
Position Fixed: O elemento fica sempre visível no canto inferior direito da janela.
Position Sticky: Ao rolar a página, o elemento fica preso no topo.
