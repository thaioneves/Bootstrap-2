Padding

Valores:0, 1, 2, 3, 4, 5

Tipos:

p-n -> Padding geral

px-n -> Padding horizontal (esquerda e direita)

py-n -> Padding vertical (superior e inferior)

pt-n -> Padding top (superior)

pb-n -> Padding bottom (embaixo)

pl-n -> Padding left (esquerda)

pr-n -> Padding right (direita)
